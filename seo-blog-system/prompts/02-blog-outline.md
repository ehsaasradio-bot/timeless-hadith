# Prompt 02 — Blog Outline
## Role: Content Strategist + Semantic SEO Architect

Using the keyword research output from Prompt 01, build a semantically rich, SEO-optimised
outline for a Timeless Hadith blog post. Apply topic cluster thinking.

---

## Input
- Keyword research output from Prompt 01
- **Word Count Target:** 1,800–2,500 words
- **Tone:** Authoritative, warm, respectful, clear — no AI padding, no filler

---

## Task

Produce a complete outline with:

1. **SEO Title** (50–60 characters) — include primary keyword naturally
2. **Meta Description** (140–160 characters) — include primary keyword + compelling hook
3. **Slug** — lowercase, hyphenated, no stop words
4. **H1** — slightly different from SEO title, rich and inviting
5. **Introduction hook** — 2 sentences describing the emotional/spiritual angle

### Content Sections (H2 / H3 structure)
- Each H2 must target a secondary keyword or LSI term
- Each H2 must include bullet notes on what to cover (2–4 sub-points)
- Include at least one H2 dedicated to authentic hadiths (with citation notes)
- Include one H2 for practical application / daily life
- Include one H2 for FAQ (target People Also Ask snippets)

6. **Internal Links** — flag 3–5 pages on timelesshadith.com to link to (use internal-links.csv)
7. **CTA** — what action should the reader take at the end?
8. **Featured Snippet Target** — which question should this post aim to own?

---

## Output Format

```
SEO TITLE: ...
META DESCRIPTION: ...
SLUG: /blog/...
H1: ...
INTRO HOOK: ...

OUTLINE:
  H2: ...
    - Sub-point
    - Sub-point
  H2: ...
    ...

INTERNAL LINKS: ...
CTA: ...
FEATURED SNIPPET TARGET: ...
```

---

## Rules
- No section should exist only to pad word count.
- Every H2 must have a clear reader benefit.
- Arabic terms must include transliteration and brief English definition on first use.
