================================================================================
PUBLIC API
================================================================================

window.SEOAudit.run(progressCallback)
  - Execute full audit across all 9 pages
  - Optional progress callback: (current, total) => void
  - Returns: Promise<auditResults>

window.SEOAudit.getResults()
  - Get last audit results from localStorage
  - Returns: auditResults | null

window.SEOAudit.getTokensUsed()
  - Get tokens consumed this week
  - Returns: number (0-5000)

window.SEOAudit.getWeeklyTokens()
  - Get detailed token info
  - Returns: { used, limit, remaining, weekStart }

window.SEOAudit.reset()
  - Clear audit results from localStorage
  - Returns: boolean

window.SEOAudit.getRecommendations(auditResults)
  - Generate sorted recommendations
  - Returns: array of { page, priority, recommendation, details }
